# 📋 Task Tracker App

A simple, modern, and efficient Task Tracker application built using **Swift**, **SwiftUI**, and **SwiftData**.  
It helps you manage daily tasks, track progress, and stay productive with a clean UI and smooth performance.

---

## ✨ Features

- Add new tasks easily
- Mark tasks as completed / undone
- Delete tasks
- Live progress tracking (completion percentage)
- Persistent local storage using SwiftData
- Smooth reactive UI with SwiftUI
- MVVM architecture for clean code structure
